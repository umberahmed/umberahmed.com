July 30. Navigation Bars

Yesterday, I practiced creating navigation bars. In the html file, I wrote out the navigation items in the form of a list:

  • About Our Farm
  • Buy Chickens
  • Chicken Feed?
  • How We Raise Our Chickens

The navigation bar was a simple bulleted list on the top of the webpage at this point. I couldn’t figure out how to get rid of the bullets or how to align each list item horizontally. I started to look for other tutorials. One suggestion from freecodecamp was to list the "a" tags in the HTML instead of creating a list. I researched a bit more and found that creating a list was common practice, so I ditched the freecodecamp tutorial and used W3 School’s CSS navigation bar concepts.

I moved from the html file and into the css file to add all the element that would create the navigation bar. I used the following css elements on the list items:

  • color: grey; /* changed foreground/text color to grey*/
  • display: inline-block; /* bullets disappeared and text aligned horizontally */
  • font-size: 12; /* font-size to 12*/
  • padding: 0 5px; /* padding of 5px to the right and left of list items*/

Above image is of progress up to the elements I’ve written about above. The navigation bar is pretty static, it doesn’t scroll and I can’t hover over the items or click on them.

Elements I decided to add were:

  • hover color
  • hover rounded edges
  • scroll with page

Beginning with the hover color, I used the list elements to add the style of white-background hover color. 

        nav li: hover {
          Background-color: white;
        }

An interesting observation here, when I would hover over the items with my mouse I could see that the padding I had initially set was displayed on only the left side of the list item. I went into the CSS file and corrected the padding dimensions so that there was padding on both sides of the list items.

Next, I noticed that the default hover-highlight was a square around the item. I wanted the hover to display nice rounded edges. I achieved this by adding a border radius of 4px to the list items in the css file and it rounded out the border! Wohoooo!!!

Finally, I was able to position the navbar as fixed to the top of the page. If I had a full webpage, a fixed position would allow the navbar to stay fixed on top of the page.
